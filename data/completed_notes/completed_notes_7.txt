<h1>Linearization and Differentials: Applications in Approximation</h1>
<p style="color:whitesmoke;">
<strong>Linearization</strong> is a local approximation technique for functions, providing a line tangent to the curve at a specified point. If <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x)</code> is differentiable at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = a</code>, the linearization at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = a</code> is given by
</p>
<p style="color:whitesmoke;">
<code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = f(a) + f'(a)(x-a)</code>
</p>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 1.</strong>
Given <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x) = 3x^2 + 4x + 1</code> and <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a = 3</code>:
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    Compute the derivative: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(x) = 6x + 4</code>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    Find function and derivative at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 3</code>:
    <ul>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(3) = 3(3)^2 + 4(3) + 1 = 27 + 12 + 1 = 40</code></li>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(3) = 6(3) + 4 = 18 + 4 = 22</code></li>
    </ul>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    Construct the linearization:
    <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = 40 + 22(x - 3)</code>
    <br>
    Expanding: 
    <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = 40 + 22x - 66 = 22x - 26</code>
    <br>
    <strong>Result:</strong> <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = 22x - 26</code>
    </p>
  </li>
</ol>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 2.</strong>
For <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x) = \sqrt{4x + 49}</code> and <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a = 0</code>:
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    Rewrite the function as a power: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x) = (4x + 49)^{1/2}</code>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    Compute derivative via chain rule:
    <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(x) = \frac{1}{2}(4x + 49)^{-1/2} \cdot 4 = \frac{2}{\sqrt{4x + 49}}</code>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    Evaluate at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 0</code>:
    <ul>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(0) = \sqrt{49} = 7</code></li>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(0) = 2/7</code></li>
    </ul>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    Linearization:
    <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = 7 + \frac{2}{7}x</code>
    </p>
  </li>
</ol>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Differentials</strong> capture how small changes in <strong>x</strong> result in small changes in <strong>y</strong>. For a differentiable function <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">y = f(x)</code>, the differential is:
<code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = f'(x) dx</code>
where <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dx</code> and <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy</code> represent small increments in <strong>x</strong> and <strong>y</strong> respectively.
</p>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 3.</strong>
For <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">y = \frac{10x - 2}{5x - 2x}</code>:
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
      Compute the differential:
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = \frac{10x - 2}{2} dx</code>
    </p>
  </li>
</ol>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 4.</strong>
Let <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">y = \frac{1}{2}(5x + 8)</code>. The product rule is used for differentiation.
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    Compute <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy</code>:
    <br>
    <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">
      dy = \frac{5}{2} \cdot \frac{1}{2}(5x + 8)
    </code>
    <br>
    Derivative simplification sequence:
    <ul>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">\frac{dy}{dx} = \left(\frac{5x + 8}{2} \right)\frac{1}{2}(5x + 8) = \frac{5x + 8}{2 \cdot 5x + 8}</code></li>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = \frac{10x + 16}{2(5x + 8)}</code></li>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = \frac{15x + 16}{2(5x + 8)} dx</code></li>
    </ul>
    </p>
  </li>
</ol>
<!-- END_SECTION -->

<h3>Using Differentials for Approximations</h3>
<p style="color:whitesmoke;">
<strong>Two primary approaches are prevalent:</strong>
<ul>
  <li>
    <strong>Option #1:</strong> Find the <strong>linearization</strong> of <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x)</code> at a convenient point <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a</code> (usually near the value of interest).
  </li>
  <li>
    <strong>Option #2:</strong> Use differentials: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = f'(x) dx</code> to estimate changes in <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">y</code> for a small increment <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dx</code>.
  </li>
</ul>
</p>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 5. Approximate</strong> <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">\ln(1.07)</code>
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option #1—Linearization:</strong>
    <ol>
      <li>Choose <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x)=\ln x</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a=1</code> (since <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">\ln 1 = 0</code> is easily evaluated).
      </li>
      <li>Compute derivative: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(x)=1/x</code>; at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a=1</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(1)=1</code>.
      </li>
      <li>
      Linearization formula: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = 0 + 1 \cdot (x - 1) = x-1</code>
      </li>
      <li>
      Approximate at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 1.07</code>:
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(1.07) = 1.07 - 1 = 0.07</code>
      </li>
    </ol>
    Result: <strong>\(\ln(1.07) \approx 0.07\)</strong>.
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option #2—Differentials:</strong>
    <ol>
      <li>
      Start with <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x) = \ln x</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 1</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dx = 0.07</code>.
      </li>
      <li>
      Compute <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = f'(x) dx = (1/1) \cdot 0.07 = 0.07</code>.
      </li>
      <li>
      The estimate is <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(1 + 0.07) \approx f(1) + 0.07 = 0 + 0.07 = 0.07</code>
      </li>
    </ol>
    </p>
  </li>
</ol>
<p style="color:whitesmoke;">
   Both methods produce the same first-order approximation based on the tangent at <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x=1</code>.
</p>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 6. Approximate</strong> <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">e^{0.49}</code>
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option #1—Linearization near <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a = 0.5</code>:</strong>
    <ol>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x) = e^x</code></li>
      <li><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(a) = e^{0.5},\quad f'(a)=e^{0.5}</code></li>
      <li>
      Linearization: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = e^{0.5} + e^{0.5}(x - 0.5)</code>
      </li>
      <li>
      At <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 0.49</code>:
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(0.49) = e^{0.5} + e^{0.5}(-0.01) = e^{0.5}(1 - 0.01) = e^{0.5} \cdot 0.99</code>
      </li>
    </ol>
    Result: <strong><code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">e^{0.49} \approx 0.99\,e^{0.5}</code></strong>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option #2—Differentials:</strong>
    <ol>
      <li>
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a = 0.5,\quad dx = -0.01</code>
      </li>
      <li>
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(a) = e^{0.5},\quad f'(a) = e^{0.5}</code>
      </li>
      <li>
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = e^{0.5} \cdot (-0.01) = -0.01 e^{0.5}</code>
      </li>
      <li>
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(0.49) \approx e^{0.5} + (-0.01 e^{0.5}) = e^{0.5}(1-0.01) = 0.99 e^{0.5}</code>
      </li>
    </ol>
    </p>
  </li>
</ol>
<p style="color:whitesmoke;">
  For non-integer values of <strong>a</strong>, calculating <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">e^{0.5}</code> exactly may require a calculator, but the process holds for other analytic functions.
</p>
<!-- END_SECTION -->

<p style="color:whitesmoke;">
<strong>Example 7. Approximate</strong> <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">\sqrt{101}</code> <strong>using differentials and linearization</strong>
</p>
<ol>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option 1—Differentials</strong>
    <ol>
      <li>Let <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(x)=\sqrt{x}</code>. Choose <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">a = 100,\, dx = 1</code>.</li>
      <li>
        Compute <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(100) = 10</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(x) = 1/(2\sqrt{x})\</code> so <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(100) = 1/20</code>.
      </li>
      <li>
        <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">dy = f'(a) \cdot dx = (1/20) \cdot 1 = 0.05</code>
      </li>
      <li>
        Approximation: <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">\sqrt{101} \approx f(100) + dy = 10 + 0.05 = 10.05</code>
      </li>
    </ol>
    </p>
  </li>
  <li>
    <p style="color:whitesmoke;">
    <strong>Option 2—Linearization</strong>
    <ol>
      <li>
      Use linearization formula:
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(x) = f(a) + f'(a)(x - a)</code> with <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f(100) = 10</code>, <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">f'(100) = 1/20</code> and <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">x = 101</code>:
      <code style="color:#00aaff; font-family: Menlo, Monaco, 'Courier New', monospace;">L(101) = 10 + (1/20)(101-100) = 10 + 1/20 = 10.05</code>
      </li>
    </ol>
    </p>
  </li>
</ol>
<p style="color:whitesmoke;">
  Both methods yield the same numerical result and are equivalent for first-order approximations.
</p>
